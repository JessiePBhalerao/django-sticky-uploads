Documentation
-----------------------------------

Documentation on using django-sticky-uploads is available on 
`Read The Docs <http://readthedocs.org/docs/django-django-sticky-uploads/>`_.


Running the Tests
------------------------------------

You can run the tests with via::

    python setup.py test

or::

    python runtests.py
